# Prisma Obras — Web

Sitio web de Prisma Obras. Construcción, remodelación e instalaciones.

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Sube este repo a GitHub
2. Importa el proyecto en [vercel.com](https://vercel.com)
3. Vercel detecta Vite automáticamente — no necesitas configurar nada
4. Conecta tu dominio desde el panel de Vercel
